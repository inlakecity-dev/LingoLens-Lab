\# LingoLens Lab



LingoLens Lab is the experimental workspace for developing \*\*LingoLens\*\*, a cross-platform OCR and translation application.



This repository contains prototypes, experiments, and feature development before they are integrated into the main LingoLens project.



\## Goals



\- Screen OCR

\- Multi-language recognition

\- Instant translation

\- Floating translation lens

\- Windows support

\- macOS support

\- Linux support
